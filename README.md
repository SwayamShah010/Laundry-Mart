🧺 LaundryMart – Home Pickup & CareLaundryMart is a responsive, client-side web application designed to simplify scheduling and booking laundry services. Users can pick services, calculate their total bill dynamically with taxes, fill out pickup details, and receive automated email confirmations via EmailJS.🌟 Key FeaturesDynamic Cart System: Real-time adding/removing of services with immediate price and 5% GST tax calculation.Interactive Booking Form: Collects user contact details, address, pickup date, and time slot preference.Smart Date Picker: Automatically restricts pickup date selection to present and future dates.Automated Email Notifications: Integrated with @emailjs/browser to send order confirmation details directly to the service provider/client.Responsive UI: Styled using modern CSS with Flexbox and Grid, fully compatible across desktops, tablets, and mobile devices.Feedback & Toast Alerts: User feedback via floating toast popups for cart additions, removals, and form submit statuses.📂 Project StructurePlaintext├── index.html     # Page structure, forms, and service cards
